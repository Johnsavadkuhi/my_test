How the CPU Works

A CPU executes instructions through a cycle of steps:

  • Fetch: The CPU retrieves an instruction from memory at the address specified by the Program Counter (PC).
  • Decode: The Control Unit interprets the instruction and determines what operation to perform.
  • Execute: The Arithmetic Logic Unit (ALU) performs the actual computation.
  • Memory Access: If the instruction requires reading from or writing to memory, this happens now.
  • Write Back: The result of the operation is stored in a register or memory location.

This simulator demonstrates these steps with a simplified CPU that has 8 registers, 64 memory locations, and basic ALU operations.
