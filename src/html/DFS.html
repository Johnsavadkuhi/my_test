About DFS Algorithm:
Depth-First Search (DFS) is a graph traversal algorithm that uses a stack to store nodes. It starts at the root and explores as far as possible along each branch before backtracking. DFS can be implemented in three ways: Preorder, Inorder, and Postorder.
