How It Works

State: Internal data managed by a component. When state changes, the component re-renders.

Props: Data passed from a parent component to a child component. Props are immutable.

Virtual DOM: A lightweight copy of the real DOM. React uses it to determine the most efficient way to update the real DOM.

When state changes, React creates a new Virtual DOM, compares it with the previous one (diffing), and updates only the changed parts in the real DOM.
