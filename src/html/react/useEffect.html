How useEffect Works

In React, the useEffect hook lets you perform side effects in function components. It serves the same purpose as componentDidMount, componentDidUpdate, and componentWillUnmount in React classes, but unified into a single API.

useEffect Syntax
import React, { useEffect } from 'react';

function MyComponent() {
  // Effect with no dependencies - runs after every render
  useEffect(() => {
    console.log('Effect ran after render');
  });

  // Effect with empty dependency array - runs only on mount
  useEffect(() => {
    console.log('Effect ran on mount');
    
    // Cleanup function - runs on unmount
    return () => {
      console.log('Cleanup on unmount');
    };
  }, []); // Empty dependency array

  // Effect with dependencies - runs when dependencies change
  useEffect(() => {
    console.log('Effect ran because dependency changed');
  }, [dependency1, dependency2]); // Dependency array
}

When does useEffect run?

  • After every render - when no dependency array is provided
  • Only on mount - when dependency array is empty []
  • When dependencies change - when specific values are in the dependency array
